# 自定义词汇训练指南 —— 让模型识别你自己的手语词

> 无论你有没有 CE-CSL 数据集，都能训练出识别自定义词汇（如"你好"、"拿铁"）的手语模型。

---

## 两种方案速览

| | 方案 A：独立训练（推荐） | 方案 B：增量训练 |
|------|------|------|
| **适用场景** | 只需识别你自己的几个词 | 既要识别 CE-CSL 的 2000 词，又要识别你的自定义词 |
| **需要 CE-CSL 数据集** | ❌ 不需要 | ✅ 需要 |
| **需要预训练模型** | ❌ 不需要 | ✅ 需要 |
| **操作步骤** | 2 步 | 5 步 |
| **前置门槛** | 录视频即可 | 需下载/配置完整 CE-CSL 数据集 |
| **识别范围** | 仅你自定义的词 | CE-CSL 2000 词 + 你的词 |

> **如果你只是做演示、验证流程、或者只关心自己的几个词 → 选方案 A。**

---

## 你借助了项目的什么

无论哪个方案，你都不是从零开始。项目提供了：

| 组件 | 作用 | 你自己写需要 |
|------|------|:---:|
| ResNet18 特征提取器 | ImageNet 预训练，冻结，提取 512 维视觉特征 | 几百行代码 + 海量数据训练 |
| CTC 损失函数 + BiLSTM 架构 | 变长序列建模，自动对齐视频帧与文本 | 200+ 行工程代码 |
| 数据加载器 | 读取 `.npy` 特征 + CSV 标注，动态 Padding | 150+ 行 |
| 推理脚本 `demo_infer.py` | 输入视频 → 输出中文文本 | 已有，直接用 |
| Web 前后端 `Project/` | 摄像头实时识别 + API 服务 | 上千行 |

**你只需要提供**：自己的手语视频。

# 方案 A：独立训练（不需要 CE-CSL）

## 目录结构

```
项目根目录/
└── data/
    └── custom_videos/           ← 已创建
        ├── 好_01.mp4            ← 你的手语视频（gitignore 已忽略）
        ├── 好_02.mp4
        ├── 好_03.mp4
        ├── ...（每个词至少 10 段）
        ├── custom.csv           ← 标注文件
        └── features/            ← 自动生成的特征
```

## 第 1 步：录制视频 + 编写标注

### 视频要求

| 项目 | 建议 |
|------|------|
| 时长 | 2-5 秒/段 |
| 数量 | **每个词至少 10 段**（越多越好） |
| 格式 | `.mp4` |
| 背景 | 尽量简洁 |
| 手势 | 参考《国家通用手语词典》或自定义 |

### 标注文件 `data/custom_videos/custom.csv`

```csv
Number,Translator,Chinese Sentences,Gloss,video_file,Note
00001,Z,好,好,好_01.mp4,
00002,Z,好,好,好_02.mp4,
00003,Z,好,好,好_03.mp4,
00004,Z,好,好,好_04.mp4,
00005,Z,好,好,好_05.mp4,
00006,Z,好,好,好_06.mp4,
00007,Z,好,好,好_07.mp4,
00008,Z,好,好,好_08.mp4,
00009,Z,好,好,好_09.mp4,
00010,Z,好,好,好_10.mp4,
```

字段说明：

| 字段 | 说明 |
|------|------|
| `Number` | 唯一序号 |
| `Translator` | 演示者标识，可自定义 |
| `Chinese Sentences` | 对应的中文 |
| `Gloss` | 手语词，多个用 `/` 分隔 |
| `video_file` | 视频文件名（相对于 `data/custom_videos/`） |

> **多个词**：直接在 CSV 中加行即可，不同 `Gloss` 值对应不同词。

## 第 2 步：训练

```bash
cd AAA

# 2.1 提取特征（只需运行一次，视频不变无需重复）
python extract_custom_features.py

# 2.2 训练模型
python train_custom.py
```

训练完成后模型保存在 `output/ctc_lstm/custom_model.pt`。

## 第 3 步：测试

```bash
python demo_infer.py --video ..\data\custom_videos\好_01.mp4 ^
    --checkpoint output/ctc_lstm/custom_model.pt
```

预期输出：

```
预测结果: ['好']
句子: 好
```

## 以后要加新词

更新 `custom.csv` 加入新词，重新录视频，然后重新跑第 2 步即可。每次都是独立训练，不依赖外部数据。

# 方案 B：增量训练（需要 CE-CSL 数据集）

> 仅当你需要模型同时识别 CE-CSL 的 2000+ 词汇时才选此方案。

## 前置条件

| 条件 | 说明 |
|------|------|
| CE-CSL 数据集已下载 | 包含 `video/` 和 `label/` 目录 |
| 数据集路径可访问 | 默认 `D:\Aprogress\Shen\dataset\CE-CSL\CE-CSL`，或用 `CECSL_DATA_ROOT` 环境变量指定 |
| 预训练模型已训练 | `output/ctc_lstm/best_model.pt` 存在 |

## 第 1 步：完成基础训练

如果还没有预训练模型：

```bash
cd AAA

# 提取 CE-CSL 全量特征
python extract_features.py --splits train,dev,test --max-samples 0

# 训练基础模型
python train_lstm.py
```

## 第 2 步：准备自定义数据

同方案 A 的第 1 步——录制视频 + 创建 `data/custom_videos/custom.csv`。

## 第 3 步：提取自定义视频特征

```bash
python extract_custom_features.py
```

## 第 4 步：增量微调

```bash
python finetune_new_word.py
```

> 如果你的 CE-CSL 数据集路径不是默认值：
> ```bash
> set CECSL_DATA_ROOT=E:\你的路径\CE-CSL\CE-CSL
> python finetune_new_word.py
> ```

## 第 5 步：测试

```bash
python demo_infer.py --video ..\data\custom_videos\好_01.mp4 ^
    --checkpoint output/ctc_lstm/best_model_finetuned.pt
```

```
方案 A（独立训练）                    方案 B（增量训练）

你的视频                              你的视频         CE-CSL 数据集
    │                                     │                │
    ▼                                     │                ▼
ResNet18 特征                              │          ResNet18 特征
    │                                     │                │
    ▼                                     │                ▼
从头训练 CTC 模型                          │         预训练 CTC 模型
    │                                     │        (best_model.pt)
    │                                     │                │
    ▼                                     │    ┌───────────┴───────────┐

(类别数 = 你的词数 + 2)                     │    │ 混合旧数据 + 新数据训练  │
    │                                     │    └───────────┬───────────┘
    ▼                                     │                ▼
custom_model.pt                           │    best_model_finetuned.pt
只能识别你的词                              │    识别 CE-CSL 2000 词 + 你的词
```

## 常见问题

### Q: 我只用方案 A，项目的前后端还能用吗？

能。`demo_infer.py` 直接可用。如果要接入 Web 后端，将 `custom_model.pt` 路径配置到 `Project/Back/app/ctc_service.py` 即可。

### Q: 方案 A 训练要多久？

22 段视频约 2-5 分钟（GPU）/ 10-20 分钟（CPU）。

### Q: 录制视频有什么要注意的？

- 手势清晰、背景干净
- 同一个词多录几个角度和速度
- 每天光线不同的时段各录几段

### Q: 想识别连续句子（如"你好谢谢"）？

在 CSV 的 `Gloss` 列用 `/` 分隔：

```csv
00001,Z,你好谢谢,你/好/谢/谢,sentence_01.mp4,
```

视频中需包含连续多个手势的动作。
